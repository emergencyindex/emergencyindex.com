
### Project template

All of the project files need to include this [YAML](http://yaml.org/) metadata at the top of the Markdown file:

```markdown
---
layout: project
volume: 
image: 
photo_credit: 
title: 
first_performed: 
place: 
times_performed: 
contributor: 
collaborators:
  -
home: 
links: 
  - 
contact: 
footnote: 
tags: 
  - 
pages:
needs_review:
---

```
