### Markdown Syntax

Markdown is a lightweight and easy-to-use syntax for styling text.

```markdown
# Header 1
## Header 2
### Header 3

- Bulleted
- List

1. Numbered
2. List

**Bold** and _Italic_ and `Code` text

[Link](https://emergencyindex.com) and ![Image](https://emergencyindex.com/assets/img/preview.png)

> blockquote

two spaces  
after text  
will not
add return

Header 0 | Header 1
-------- | --------
cell 1 | cell 2
cell 3 | cell 4

HTML
<dl>
  <dt>Definition list</dt>
  <dd>Is something people use sometimes.</dd>

  <dt>Markdown in HTML</dt>
  <dd>Does *not* work **very** well. Use HTML <em>tags</em>.</dd>
</dl>

Horizontal Rule

---

```

The above markdown code looks like:

# Header 1
## Header 2
### Header 3

- Bulleted
- List

1. Numbered
2. List

**Bold** and _Italic_ and `Code` text

[Link](https://emergencyindex.com) and ![Image](https://emergencyindex.com/assets/img/preview.png)

> blockquote

two spaces  
after text  
will not
add return (and multiple line breaks are ignored!)

Header 0 | Header 1
-------- | --------
cell 1 | cell 2
cell 3 | cell 4

HTML
<dl>
  <dt>Definition list</dt>
  <dd>Is something people use sometimes.</dd>

  <dt>Markdown in HTML</dt>
  <dd>Does *not* work **very** well. Use HTML <em>tags</em>.</dd>
</dl>

---

For more details see [GitHub Flavored Markdown](https://guides.github.com/features/mastering-markdown/).
